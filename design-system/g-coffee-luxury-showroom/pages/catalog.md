# Catalogue Override

- Desktop composition: 3-column card grid plus 286-300 px sticky filter rail.
- Tablet/mobile: filter drawer with active count; grid 2 then 1 column.
- Cards are flat dossiers: square visual, taxonomy, name, tasting profile, indicative range, two actions.
- Search uses deferred value and provides query-aware empty guidance.
- Filters: category, origin, roast, format, availability, maximum indicative price.
- Product detail: media + information + calculator, followed by keyboard tabs and offset related products.
- Calculator exposes the formula and ends in a consultation request, not a purchase.
